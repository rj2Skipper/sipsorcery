RTPSessionStream(Int32, Int32, List<Int32>)

Creates a lightweight class to track an RTP stream within an RTP session. When supporting RFC3550 (the standard RTP specification) the relationship between an RTP stream and session is 1:1. For WebRTC and RFC8101 there can be multiple streams per session.
